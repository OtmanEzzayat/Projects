Descripción detallada

Programa principal.

Después de inicializar el Cluster y el área de Espera, podemos ir introduciendo comandos por el canal estándar de entrada para gestionar los elementos de las clases inicializadas. Asimismo, si alguna instrucción no es posible de realizar, se enviará un mensaje de error por el canal de salida y también, el mismo canal será usado para imprimir datos. Para terminar de ejecutar comandos, habrá que introducir el comando "fin".
