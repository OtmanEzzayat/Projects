Descripción detallada

Representa un procesador con los atributos identificador, memoria máxima y contiene varios diccionarios para almacenar y gestionar procesos y espacios.

Además, ofrece varias operaciones para añadir, borrar o gestionar procesos en memoria y otras operaciones para consultar datos.
